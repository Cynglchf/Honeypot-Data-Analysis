# Honeypot Data Analysis

This project analyses real attack data collected from a honeypot system.

<img width="1348" height="632" alt="Attack_Map" src="https://github.com/user-attachments/assets/ba06fc81-ad09-4e7b-940d-f9e6128f7061" />

### Project Details:

Honeypot: [T-Pot](https://github.com/telekom-security/tpotce)

Deployment: [Azure Virtual Machine](https://azure.microsoft.com)

Analysis and Visualization: [Kibana](https://github.com/elastic/kibana)

### Analysis 

All analyses can be found in the `/analysis/` directory.

Analyses include:

**Credential Analysis:** Analysis of attempted usernames and passwords.  
**Attacker Behavior Analysis:** Examination of commands and actions performed by attackers.  
**Malware Analysis:** Analysis of malware samples collected during attacks.  
**CVE Analysis:** Analysis of vulnerabilities targeted by attackers.
